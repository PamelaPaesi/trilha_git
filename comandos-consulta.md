# Comando de consulta

## cd - navegar para determiando diretório
Comando: 
```sh
cd [caminho da pasta]
```
## ls - listar arquivos e pastas de um determinado local
Comando:
```sh
ls
ls [caminho da pasta]
```
## cd ~ - navegar para a pasta do usuário logado 
Comando:
```sh
cd ~
```
## clear - limpar terminal
Comando:
```sh
clear
```
